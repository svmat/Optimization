2. Passing by value, reference, pointer
3. Constructing/Destructing objects on stack, heap.
4. Cache hit/miss
5. Optimizing code by changing data types
6. Return value optimization
7. Proxy
8. Compiler settings
9. Usage of inline functions
10. Operator overloads
11. SIMD usage
